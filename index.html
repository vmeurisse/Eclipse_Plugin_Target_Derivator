Eclipse plugin target derivator

Eclipse plugin to mark maven target folders as derived resources

Download

This plugin add a button to your eclipse to mark as derived several folders in just one click. The folders ar selected by their name:

  • target
  • node_modules
  • output
  • bin

To install the plugin, copy the latest jar in your Eclipse dropins/ folder and restart Eclipse. You will have a new Derivator menu in the main toolbar
